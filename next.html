Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM: 1. getElementById(), 2. getElementsByTagName(), 3. querySelector(), 4. querySelectorAll(), 5. getElementsByClassName(), 6.getElementsByName().

Question-2: What is the difference between innerHTML, innerText, and textContent?

The difference between innerHTML, innerText and textContent is that innerHTML gets or sets the HTML content inside an element, innerText gets or sets the visible text inside an element excluding any hidden text whereas textContent gets or sets the full text content including any hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element handles events for its child elements dynamically even if they are added later.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event starts from the target element and propagates (bubbles) up to its parent elements. if we click a button inside a div, the button will be clicked first, then the parent div will be clicked as well.

Question-5: How do you create, add, and remove elements using JavaScript?

Using JavaScript, we use document.createElement() method for creating an element, document.body.appendChild() method for adding an element and document.body.removeChild() for method for removing an element.
